# PureVibes

A Super Hexagon-inspired survival game. Dodge collapsing hexagonal walls by rotating around the center. Pure vibe coded — no manual code allowed.

## Run locally

The game is vanilla HTML/JS with no build step. Just serve the files with any static server:

```bash
python3 -m http.server 8080
```

Then open [http://localhost:8080](http://localhost:8080) in your browser.

To stop the server, press `Ctrl+C` in the terminal.

## Controls

| Key | Action |
|---|---|
| `A` / `Arrow Left` | Rotate left |
| `D` / `Arrow Right` | Rotate right |
| `Space` | Start / Restart |
| `Escape` | Back to title |
| `M` | Toggle mute |
